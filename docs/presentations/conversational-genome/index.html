Try the demo
Ask the genome above. Every answer is an executed skill with provenance, confidence, and an ancestry-validity flag.

Run skills on your own data
Install ClawBio and run the same PGx skill on your genotypes, locally and reproducibly.
pip install clawbio
clawbio run pharmgx --demo
clawbio run pharmgx --input your_genotypes.txt
Run your first skill →
Integrate into your agent
Call versioned skills from your own agent and get structured, auditable output.
from clawbio import run_skill
result = run_skill("pharmgx",
    input="your_genotypes.txt")
# skill@version, CPIC guideline, checksum

First-run paths

Find your fastest first success

Clinical PGx implementer
  1. Ask the demo about warfarin, then open Audit on the card.
  2. Read the skill version, the CPIC guideline, and the reproduce command.
  3. pip install clawbio, then run it on a test genotype file.
  4. Open the PharmGx skill doc (13 genes, 59 drugs, CPIC mapping).
  5. Pin the skill version in your pipeline for reproducibility.
First success: a reproducible PGx report on your own data.
Researcher / biobank PI
  1. Ask the demo how reliable this is for Peruvian ancestry.
  2. See the 72 / 51 / 40 ancestry transfer and the evidence behind it.
  3. Run PharmGx across a cohort export; read the per-call ancestry flags.
  4. Quantify the gap with the Equity Scorer (HEIM) skill.
  5. Cite the versioned-skill benchmark and pin versions in your methods.
First success: an ancestry-stratified, reproducible cohort read.
Developer
  1. Try the demo; note that answers are executed skills, not free text.
  2. pip install clawbio, then clawbio list.
  3. from clawbio import run_skill inside your own agent.
  4. Read Building custom agents.
  5. Build a skill and add it to the library.
First success: your agent answers by executing a versioned skill.
